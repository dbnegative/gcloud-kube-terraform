# Google Cloud + Kubernetes + Terraform 
 
Automated provisioning of kubernetes based off https://github.com/kelseyhightower/kubernetes-the-hard-way

Requires the following: 
* Terraform 7rc1 
* Google Cloud SDK
* Fully functioning service account creds
